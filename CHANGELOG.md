# 更新日志

## v1.1.0 (2025-12-24)

### ✨ 新功能
- **密码鉴权**: 新增访问密码保护，可通过环境变量 `VITE_AUTH_PASSWORD` 配置
- **信件 emoji Favicon**: 页面图标更新为信封 ✉️ emoji
- **部署工作流**: 新增 `.agent/workflows/deploy-letter.md` 工作流，支持一键处理新信件并部署

### 🔧 优化
- **配置文档**: README 新增环境变量配置说明
- **项目结构重构**: System 目录迁移至 `.agent/rules/`，提示词和规则更加规范
- **凭据管理**: 新增 `.agent/secrets.env` 用于安全存储服务器凭据（已被 gitignore 忽略）

### 🛡️ 安全
- 所有敏感信息（密码、API Key、个人档案）已从仓库中移除
- 工作流文件中的凭据使用占位符
