# 更新日志

## v1.2.1 (2025-12-24)

### 🐛 修复
- **桌面端布局回归**: 修复了在 v1.2.0 中由于过度强调移动端叠层架构，导致桌面端三栏布局中信件详情页无法显示的 Bug。现在应用能完美识别设备类型，在 Desktop 模式下自动切换回 Flex 响应式布局。

## v1.2.0 (2025-12-24)

### ✨ 新功能
- **深度移动端适配**: 重构应用架构，支持手机端高性能交互
  - **Apple 风格叠层导航**: 实现类似 iOS 的模态弹出层级，详情页全屏弹出并完全覆盖导航栏
  - **极速响应动效**: 极致调优物理引擎参数，实现零延迟感的「瞬时」交互反馈
  - **灵动底部导航**: 新增磨砂胶囊风格导航栏，支持具有空间感（方向性）的切屏动画
- **密码鉴权**: 新增访问密码保护，可通过环境变量 `VITE_AUTH_PASSWORD` 配置
- **信件 emoji Favicon**: 页面图标更新为信封 ✉️ emoji
- **部署工作流**: 新增 `.agent/workflows/deploy-letter.md` 工作流，支持一键处理新信件并部署

### 🔧 优化
- **交互联动**: 实现从笔友列表选中后自动带动画平滑跳转至时光轴的逻辑
- **视觉适配**: 优化移动端纸张边距、附件网格展示，并修复了胶囊导航对页脚的遮挡问题
- **项目结构重构**: System 目录迁移至 `.agent/rules/`，提示词和规则更加规范
- **凭据管理**: 新增 `.agent/secrets.env` 用于安全存储服务器凭据（已被 gitignore 忽略）

### 🛡️ 安全
- 所有敏感信息（密码、API Key、个人档案）已从仓库中移除
- 工作流文件中的凭据使用占位符
